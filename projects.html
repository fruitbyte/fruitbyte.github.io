Projects

Here are some extracurricular projects that I made, organized into unmanned vehicles, development, and multidisciplinary projects. If you want to learn more about any project, feel free to reach out and I'd be happy to have a conversation!

Unmanned Vehicles

Unmanned Aerial Vehicles (UAVs), Unmanned Ground Vehicles (UGVs)

Scorpion UGV

Scorpion is the second generation FPV Beyond Visual Line-Of-Sight ground vehicle, designed in SolidWorks CAD software. This UGV features a retracting arm system that picks up packages as well as performing all the other functions of Rhino. Scorpion was built to compete in the Unmanned Systems Canada 2021 Competition.

The Project Boom

I am currently working on the wing structures of the 3D Printed Concept Plane for The Project Boom, a group of engineering students aiming to be the first to break the sound barrier with RC aircraft and break the world record. I use SolidWorks CAD and FEA help determine internal wing structure and spar placement.Website

VTOL Design

I designed a Vertical Takeoff and Landing flying wing using XFLR5 for preliminary airfoil analysis, simulating airflow at low Reynold's numbers and low angle of attack to tune stability characteristics. The rest of the aircraft was designed in SolidWorks and the hardware used makes use of the components in my 250 drone project.

Autonomous Model Train

A line-following train using IR sensors that plays "We Wish You A Merry Christmas" as it moves along the track. It is all driven from an Arduino Uno connected to IR sensors, an SD card reader breakout board, and a small speaker. Documentation

Rhino UGV

Rhino is a FPV (first person view) remote controlled Beyond Visual Line-Of-Sight (BVLOS) ground vehicle built to scan and retrieve medical packages in remote locations. Rhino was designed in OnShape CAD. Video

250 Quadcopter

My first DIY drone project powered by a STM32F405 flight controller, 4 brushless DC motors, and a 3S LiPo. This drone was built to be a testbed for future UAV experiments.

Development

Coding projects: AR, websites, etc.

Autobud Mobile Application

A flutter application that uses phone GPS data with car information to provide vehicle idle impact results. Information is stored and served from a Firebase Cloud Firestore Database and features OAuth2 authentication. This application has placed as an international semifinalist in the global Technovation Challenge. Website

Uniform Marking Android Application

An android application developed with Firebase Realtime Database (JSON object database), featuring ordered data queries, exception handling, object-oriented programming, and intents. Developed exclusively for the client 810 Royal Canadian Air Cadet Squadron that replaced a pen-and-paper system which reduced uniform marking speeds by roughly 15% while saving paper.

Project Boom WebAR

A web-based Augmented Reality viewer for The Project Boom, for the sharing and promotion of the supersonic aircraft design. The AR demo is a javascript implementation of WebAR surface recognition and is compatible with iOS 11+, Android 9+, IE11+, and Chrome. AR Demo

UBC UAS AR Model Viewer

This model viewer for visualization purposes was built off of the same source code as the Project Boom WebAR project I worked on. This visualization tool helped with taking off the CAD goggles and seeing how things scale in real life!

Mike Hofer Music Website

A website developed for the client, Mike Hofer, using HTML, Bootstrap CSS, and Javascript. See at mhmmikehofermusic.ca

Crazy Eights

A Crazy Eights game created with the JavaFX GUI library where the player plays against an NPC. The game is downloaded and ran as a .jar.

Multidisciplinary Projects

Other projects I had a lot of fun with!

Studio

A studio the size of a tiny house, featuring a cantilevered design and full electrical. My largest project to date, designed and hand built from the ground up by myself and a few friends.

Leap Motion Hand Controlled Robotics

Remote hand-controlled robotics was the focus of this project, using the Leap motion tracker connected via Bluetooth. I wrote programs for hardware (Arduino Mega connected to 5 servos in a hand/arm configuration) involving TX/RX, Java data acquisition from the meshed IR camera module, and designed the schematic.

P39 Airacobra Restoration

As an Assistant Restoration Technician at the Alberta Aviation Museum, I used air and power tools to fabricate and restore the sheet metal skins and internal structures of two deteriorated P39 Airacobras from engineering drawings.
